F. Rincón; J.D. Dondo; J. Barba; J. Caba; F. Sánchez Molina
Cenference: Jornadas de Computación Empotrada

Abstract
Una de las estrategias fundamentales para aumentar la productividad del dise˜no es el reuso de soluciones previas, generalmente cores IP con cierto grado de configuración. La flexibilidad de esta aproximación, sin embargo, es limitada, ya que los cores están descritos a un nivel muy cercano a la implementación, y por lo tanto optimizados para un determinado tipo de contexto. Por otro lado los sistemas empotrados demandan cada vez más un mayor grado de conectividad, lo que se traduce en la necesidad de incoporar circuitería específica para el manejo de las comunicaciones. La solución que se propone en este trabajo consiste en el uso de una librería de comunicaciones descrita en un lenguaje de alto nivel, y en el uso de herramientas de síntesis de alto nivel para componer y generar el controlador de comunicaciones adecuado para cada situación. La librería incorpora además los bancos de prueba necesarios para facilitar la verificación automática del controlador hasta el nivel de implementación.

Network Enhancement: providing virtual QoS over existing networks
G. Urzaiz; D. Villa; F. Moya; J.C. López
Cenference: International Symposium on Ubiquitous Computing and Ambient Intelligence
Location: Riviera Maya (México)
Date: 05/12/2011 - 09/12/2011
ISBN: 978-84-694-9677-0
Abstract
Better applications require enhanced networking, specially in highly demanding applications such as in the case of modern Ambient Intelligence (AmI). Unfortunatelly the network is there and should make the most possible so that not be a barrier. A new Network Enhancement concept is proposed, defined as the action and effect of enhancing the network. This approach may be useful to set specific enhancement objectives and to make sure that the final effect in the network is achieved. In this paper, the Network Enhancement concept is described, and the Virtual Quality-of-service Network (VQN) model is presented as an example of the development of an external network enhancement tool to provide virtual QoS over an existing network.
